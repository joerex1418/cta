# cta
A Python wrapper for interacting with the CTA Transit Feeds:  

**BusTrackerAPI**  
**TrainTrackerAPI**  
**CustomerAlertsAPI**



## Getting Started
```python
import cta

# API Wrappers
busapi = cta.BusTracker()
trainapi = cta.TrainTracker()

# Wrappers for Bus Objects
```

## BusTracker Objects
- BusRoute
- BusStop
- Bus


## TrainTracker Objects
- TrainRoute
- TrainStation
- Train
